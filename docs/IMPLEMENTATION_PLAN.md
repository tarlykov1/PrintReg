# План реализации GSPLabelPrinter

1. Создать solution с WinExe ASP.NET Core Minimal API на `net8.0-windows` и тестовый проект.
2. Разнести backend на сервисы конфигурации, CSV, резервных копий, поиска, печати, блокировки запуска и очистки логов.
3. Реализовать локальные API только на `127.0.0.1:5187` и статический frontend без внешних зависимостей.
4. Реализовать безопасное добавление сотрудников через временный CSV, проверку чтения и замену файла после резервного копирования.
5. Реализовать Windows-печать через `System.Drawing.Printing` с размером бумаги 40 × 60 мм.
6. Покрыть ключевые сценарии автоматическими тестами и добавить скрипт self-contained публикации win-x64.
